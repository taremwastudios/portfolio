# Portfolio Website

A static portfolio website built with HTML, CSS, and JavaScript.

## Deployment on Koyeb

This project is configured for deployment on Koyeb as a static site.

### Configuration Files

- **package.json**: Defines the project and includes the `serve` package to serve static files
- **koyeb.yaml**: Koyeb-specific configuration for deployment

### Build and Run

- **Build Command**: `npm install` (installs dependencies)
- **Run Command**: `npx serve -s . -l 8080` (serves static files on port 8080)

### Local Development

To run locally:

```bash
npm install
npm start
```

Then open http://localhost:8080 in your browser.

## Files

- `index.html` - Main HTML file
- `styles.css` - CSS styles
- `script.js` - JavaScript functionality
- `me.jpg` - Profile image
- `icons.png` - Icons image
